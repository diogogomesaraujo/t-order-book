## About

This project explorates STM as a concurrency model for building composable concurrency in functional programming languages. It was implemented as the final project for the Advanced Topics in Functional Programming course at FCUP. To learn more about STM and the case study developed read the following [report](./TAPF.pdf) or the [presentation's slides](./slides/main.pdf).
